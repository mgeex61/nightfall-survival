# Nightfall Survival

A top-down survival game built with **HTML, CSS, and JavaScript**.

## Features

- Large scrolling outdoor map
- Multiple houses to enter
- Multi-floor house interiors
- Day and night cycle with clock
- Auto loot collection
- Barricading doors and windows
- Zombies outside and inside
- Pistol and shotgun
- Melee attack
- Inventory overlay
- Hunger, health, food, wood, and medkits

## Controls

- **W / A / S / D** — Move
- **Mouse** — Aim
- **Left Click** — Shoot
- **F** — Enter / Exit house / Switch floors
- **E** — Eat / Use medkit / Barricade
- **X** — Remove barricade
- **Q** — Melee attack
- **1** — Select pistol
- **2** — Select shotgun
- **I** — Open / close inventory

## How to play

Survive as long as possible.

During the day, explore the map, collect loot, and prepare for the night.  
At night, more zombies appear, and houses become much more important for survival.

Use food to manage hunger, gather wood to barricade doors and windows, and switch between weapons depending on the situation.

## Tech

- HTML5 Canvas
- CSS3
- Vanilla JavaScript
